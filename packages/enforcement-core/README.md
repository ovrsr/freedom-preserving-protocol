# @ovrsr/fpp-enforcement-core

Harness-agnostic enforcement engines for the Freedom Preserving Protocol
(classifier, disposition engine, mandate store, receipt/audit helpers).

Depends on an **exact** `@ovrsr/fpp-protocol-core` pin. No OpenClaw peer.

## Install

```bash
npm install @ovrsr/fpp-enforcement-core@1.0.0
```

## Development

```bash
npm run build -w @ovrsr/fpp-enforcement-core
npm test -w @ovrsr/fpp-enforcement-core
```
