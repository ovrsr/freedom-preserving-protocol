# @ovrsr/fpp-trust-core

Harness-agnostic trust graph, handshake, quorum session, disputes, and capsules
for the Freedom Preserving Protocol.

Depends on an **exact** `@ovrsr/fpp-protocol-core` pin. No OpenClaw peer.

## Install

```bash
npm install @ovrsr/fpp-trust-core@1.0.0
```

## Development

```bash
npm run build -w @ovrsr/fpp-trust-core
npm test -w @ovrsr/fpp-trust-core
```
