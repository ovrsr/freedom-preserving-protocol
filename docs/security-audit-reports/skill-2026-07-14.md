Overview
The skill is mostly transparent and purpose-aligned, but it includes optional broad tool-interception hooks and unattended enforcement paths that deserve manual review before installation.

Install the prompt-layer skill only if you want an agent-governance framework that can append local memory/audit records after explicit adoption. Do not install or merge the adapter hook configs unless you intentionally want FPP code to run before tool calls and potentially allow, deny, or ask for approval based on its policy. Review FPP_ENFORCEMENT_CONFIG, audit-log paths, and any standing mandates before enabling unattended mode.
